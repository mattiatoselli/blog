Virtualbox, macchine virtuali: cosa sono?

Virtualbox è un software progettato dalla Oracle sotto licenza GPL (Gnu Public License) nella sua versione base. Grazie ad esso è possibile creare una macchina virtuale sul proprio computer, in modo da utilizzare un altro sistema operativo dalla nostra macchina. Una macchina virtuale è un ambiente virtuale creato utilizzando un apposito software che emula il comportamento di una macchina fisica (client o server). La creazione della macchina virtuale è possibile destinando temporaneamente ad essa una parte delle risorse hardware della macchina fisica (RAM, CPU, GPU, hard disk/SSD). Ogni volta che una macchina virtuale (sistema "guest") viene spenta, tutte le risorse vengono automaticamente liberate e restituite alla macchina ospitante (sistema "host") fatta eccezione per la totalità o una parte dei dati memorizzati negli hard disk virtuali. Le informazioni relative al sistema operativo e ai programmi installati installati nella macchina virtuale (così come i dati in essa salvati) vengono memorizzate e conservate sul disco fisso o sull'unità SSD del sistema host.

Perchè ci serve Virtualbox?

Avere a disposizione direttamente un server su cui lavorare è un lusso che non capita quasi mai. Inoltre, una delle best practise maggiormente apprezzate per un professionista serio, consiste nel seguire certe metodologia di sviluppo:
1) Sviluppare le nuove funzionalità richieste dal cliente in locale.
2) Migrarle in un ambiente di test dove il cleinte possa provarle.
3) Portare le modifiche in ambiente di produzione quando esse abbiano raggiunto un adegato livello di stabilità e maturità.
Poichè in questo corso vogliamo imparare a sviluppare seguendo tutte le best practise, seguiremo questa strada. Ricordate: le best practise non sono un esercizio di stile, sono abitudini che rendono più facile lavorare a noi, e più piacevole modificare il nostro codice per i programmatori che arriveranno dopo.

Installare Virtualbox

Per scaricare il software basterà recarsi a questo link e selezionare la versione adatta al proprio sistema operativo. Esistono versioni per tutti i sistemi operativi, in alternativa, per distribuzioni linux basate su Debian come Ubuntu, oltre che dal proprio centro software, è possibile lanciare da terminale il comando:

sudo apt install virtualbox

Scaricare Vagrant

Vagrant è un gestore di macchine virtuali, in sostanza, grazie a Vagrant è possibile costruire delle macchine virtuali che utilizzano le stesse configurazioni e possono essere create, modificate e cancellate con facilità. Una particolarità di Vagrant è che ogni sviluppatore può creare la propria macchina virtuale, e condividerla col mondo intero in modo del tutto open source. Ogni sviluppatore potrà scaricare queste macchine virtuali, dette "Box", e utilizzarle per i propri scopi. Il vantaggio rispetto ad altri gestori di macchine virtuali come docker, è che Vagrant da una netta separazione tra quella che è la macchina che hosta il server di sviluppo, ed il server stesso.

Scarichiamo Vagrant a questo link. Nel caso di distribuzioni Linux, scaricato il file apriamolo con un gestore di pacchetti adatto alla nostra distribuzione. Per sicurezza, una volta completata l'installazione, riavviamo il computer.

Installare Git

Git è un sistema di versionamento del codice, molto apprezzato da tutti gli sviluppatori. Git venne sviluppato da Linus Torvalds agli albori di Linux, per permettere al suo team di sviluppo di modificare e versionare velocemente il codice del kernel del sistema operativo open source. Da allora, di acqua sotto i ponti ne è passata, e Git si è imposto come un sistema di versionamento del codice particolarmente intuitivo ed utile. Tutte le macchine linux dovrebbero avere git già preinstallato, una rapida verifica digitando "git" nel terminale ci dirà se è così, altrimenti, digitare il comando indicatoci dal terminale risolverà il problema. per quanto riguarda Mac, l'installer grafico è disponibile qui.
Anche in questo caso, però dovrebbe già essere preinstallato. Per Windows basta andare qui, una volta scaricato lanciare l'eseguibile e seguire le istruzioni.Scaricate tutte le estensioni disponibili, per il resto lasciate tutte le configurazioni di default.
